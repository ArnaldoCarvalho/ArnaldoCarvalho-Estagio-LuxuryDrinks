Desenvolvimento de API

Desenvolvimento de uma API em Java destinada inicialmente às aplicações mobiles a serem desenvolvidas e também para realização de futuras integrações com os sistemas informáticos da empresa. Automação entre o SQL Server e a Zone Soft

Desenvolvimento de uma automação que realiza a atualização das guias de consignação na Zone soft. Para resolver isto decidimos fazer conexão direta ao SQL Server da empresa, evitando assim erros de passagem de informação. Para realizar a conexão à Zone Soft utilizamos o selenium, um emulador do navegador, o que permite controlá-lo. Com esta abordagem conseguimos atualizar as guias de consignação automaticamente sem depender da intervenção humana, o que reduz erros e tempo em tarefas repetitivas.

📚 Competências Desenvolvidas

  • ✔ Automatização de processos
  • ✔ Integração entre sistemas
  • ✔ Selenium WebDriver
  • ✔ Desenvolvimento com Python

Aplicação para Gerar Listas de Preços

Desenvolvimento de uma aplicação que gera listas de preços em formato pdf e excel. Esta aplicação desenvolvida em python, faz diversas consultas ao SQL Server da empresa e permite de forma rápida e eficiente consultar artigos ou retirar diversas listas de preços em formato pdf ou excel, com filtros aplicados. Enquanto o ERP atual da empresa demora entre 30min a 2h para obter várias listas, o novo sistema demora menos de 5 min obter o mesmo resultado.

📚 Competências Desenvolvidas

  • ✔ Desenvolvimento Desktop
  • ✔ Manipulação de ficheiros Excel e pdf
  • ✔ Automatização de processos
  • ✔ Interfaces para utilizador
  • ✔ Processamento de dados
